/*************
Enumerations :
*************/

Essentially a list of constants. 

In Java, an enumeration defines a class type.

enum Apple {
	Jonathan, GoldenDel, RedDel, Winesap, Cortland
}

The apple types are enumeration constants, each is implicitly a public static final member of class Apple. 
You can create a variable of type <some numeration>, it's values will be restricted to the enum's constants.

e.g. Apple a = Apple.RedDel;

Comparisons are dones as primitives 
if(a == Apple.GoldenDel){
	...
}

And can be used in switch statements 

// Use an enum to control a switch statement.
switch(ap) {
 case Jonathan:
 // ...
 case Winesap:
 // ..

values() returns a list of enumeration constants
valueOf(<string>) returns a particular enumeration constant which matches the string passed in
